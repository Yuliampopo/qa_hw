1)  'number' + 3 + 3
Плюс бачить строку і тому усе конкатинує і видає number33
2)  null + 3
Буде 3, null стає 0
3)  5 && "qwerty"
Логічне && (AND) вичислює операнди 5 та "qwerty" обидва вони true , в такому випадку повертає останню, тобто qwerty
4)  +'40' + +'2' + "hillel";
Перші два числа сумує, а потім конкатинує до стрінги і видає 42hillel
5)  '10' - 5 === 6;
Рядок стає числом, 10 - 5 = 5 і воно не дорівнює 6, тому false
6)  true + false
true це 1, false це 0 це тому результат 1
7)  '4px' - 3
Буде NaN, бо не може зробити число
8)  '4' - 3
Рядок перетвоює на число і видає 1
9)  '6' + 3 ** 0;
3 возводить в ступінь 0, що дорівнює 1, а потім бачить строку і конкатенує 6 та 1, виходить 61
10)  12 / '6'
Строку робить числом і ділить, видає 2
11)  '10' + (5 === 6);
5 не дорівнює 6, тому це false, потім бачить строку '10' і конкатенує усе, видає 10false
12)  null == ''
видає false, пуста строка це не null
13)  3 ** (9 / 3);
Робить ділення в дужках і там ми отримаємо 3, потім іншу 3 возводить в 3 ступінь, видає 27
14)  !!'false' == !!'true'
Тут два заперечення, з одним знаком ! буде false, з !! буде true і там і там, тому true
15)  0 || '0' && 1
Перший у нас 0 це false, нуль в лапках це стрінга і це true, результатом першої операції буде '0', далі '0' і 1, так як обидва true, то результатом буде те, що справа, тобто 1
16)  (+null == false) < 1;
В дужках порівняння true, бо там нулі порівнюються, true це 1, а один не більше 1, тому false
17)  false && true || true
false та true дає false, false або true дає true
18)  false && (false || true);
false або true це true, false і true це false
19)  (+null == false) < 1 ** 5;
В дужках порівняння true, бо там нулі порівнюються, true це 1, 1 в 5 ступені це 1, 1 не більше 1, отже false
